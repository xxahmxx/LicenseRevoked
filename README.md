[ EN ] ENGLISH INSTRUCTIONS
-------------------------------------------------------------------

⚠️ WARNING: This is a "Permadeath" mod for your driver's license!
If you exceed 24 penalty points, your profile will be LOCKED.
Drive carefully!

>>> INSTALLATION (Easy Method)
1. Open your game installation folder:
   - Euro Truck Simulator 2  OR  American Truck Simulator
2. Open this ZIP archive.
3. Drag and drop the "bin" folder from this ZIP directly into your main game folder.
   (If Windows asks to merge folders, click YES).
4. Start the game.
5. Click "OK" on the SDK Warning ("Request to use advanced SDK features").

>>> RULES
- Limit: 24 Points. Exceeding it results in immediate GAME OVER.
- Saving: Points are saved permanently per truck/profile.
- Reset: To clear your points, you must drive for 14 in-game days without ANY violation.

>>> HOW TO UNBAN (Emergency)
If you get banned and want to play again on the same profile:
Go to "bin/win_x64" folder and delete the file named "lr_save_YOUR_LICENSE_PLATE.txt".
